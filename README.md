# Chromicle
Chromicle is a Google Chrome history extension for better display and searching.

# Feature
- [x] Better display
- [ ] Pagination
- [x] Search by keyword
- [x] Search by date range
- [x] Delete by date range
- [ ] Delete by selection

# How to

```
yarn
```
For develop (need to refresh manually)
```
npm run watch
```
For production
```
npm run prod
```
